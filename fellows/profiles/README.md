# Fellow Profiles

This folder contains Humanitarians AI fellow profile videos.

Use one lowercase-kebab folder per fellow:

```text
fellows/profiles/first-name-last-initial/
```

Each profile folder should contain a `beat_sheet.json`, README, sources,
fact-check record, build prompt, and explicit review and publishing gates.
Fictional examples must say they are fictional. Real profiles must use only
verified names, affiliations, accomplishments, quotations, and public links.

Each fellow chooses one Kokoro voice before their first video and keeps it
across their profile and weekly report series. With no stated preference, use
an `af_*` suggestion for a female-coded name and an `am_*` suggestion for a
male-coded name. The fellow's explicit preference always overrides the
name-based suggestion.

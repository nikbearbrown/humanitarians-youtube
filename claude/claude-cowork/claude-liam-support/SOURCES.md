- Response drafting draws on configured tone + knowledge base + the specific request. ✓ (source)
- Knowledge-base building turns support history into a structured FAQ that reduces future volume. ✓ (source)
- Sentiment detection flags frustrated / at-risk customers for personal attention. ✓ (source)
- Configuration: tone, escalation triggers, VIP identification — set what routes to a human. ✓ (source)
- Principle: draft, don't autosend; review every response before it reaches a customer. ✓ (source)
- Principle: recurring support requests are product feedback in disguise. ✓ (source)
